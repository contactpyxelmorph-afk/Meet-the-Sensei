// 001 · Venture

Pyxel Corp

⬡ Solo Indie Studio

8 production tools built for the GB Studio game-dev community

Pyxel Corp is a one-person software studio producing specialised desktop tools for developers working with GB Studio — the GameBoy Color-style engine. All 8 tools were designed, coded, and shipped between December 2024 and February 2025, demonstrating rapid end-to-end product delivery. Each ships as a standalone Windows executable with zero runtime dependencies.

Stack: Python · Tkinter / CustomTkinter · Pillow · Flutter/Dart · HTML/CSS · PyInstaller

⬡ PYXEL MORPH

Converts input images into green-shaded GBC palette previews, auto-generating a MAPPING FILE and PALETTE LIST that drop directly into GB Studio. Eliminates the need to build green-shade references by hand. Supports 2 priority colours per tier on 8×8 tile grids.

PythonImage ProcessingGB StudioPalette Mapping
→ GitHub

⬡ PYXEL CLEAR

Solves "Palette Bloat" in GB Studio by forcing new scenes to reuse existing palette libraries rather than stacking near-identical duplicates. Uses mathematical best-fit tile-assignment on 8×8 grids — consuming zero new palette slots.

PythonPalette OptimisationScene QuantisationGB Studio
→ GitHub

⬡ PYXEL TEXT

Standalone dialogue editor for GB Studio. Extracts all text events (dialogues, menus, choices) across every scene into a searchable interface — no need to open the full engine. Multi-scene sessions with in-memory state; changes persist until export.

PythonDialogue EditingMulti-SceneGB Studio
→ GitHub

⬡ PYXEL MATRIX

Generates and visualises GB Studio tile-matrix layouts. Automates the tedious mapping of tile coordinates across large scenes, giving developers a clear spatial overview of their GBC scene structure.

PythonTile MatrixScene LayoutGB Studio
→ GitHub

⬡ PYXEL PALETTE

Dedicated palette builder and inspector for GBC-style projects. Creates, edits, and exports .hex palette files compatible with GB Studio's format, streamlining the colour definition workflow.

PythonPalette BuilderHEX ExportGB Studio
→ GitHub

⬡ PYXEL TILE

Tile slicing and analysis tool. Breaks large background images into the 8×8 tile grid required by the GBC hardware constraint, identifying unique tiles and flagging duplicates to assist optimisation.

PythonTile SlicingDeduplicationGBC Constraints
→ GitHub

⬡ SPRITEBINGO

Sprite-sheet validation tool. Cross-references a game project's sprite definitions against the actual image assets, surfacing mismatches and missing entries before they cause runtime issues in GB Studio.

PythonSprite ValidationAsset PipelineGB Studio
→ GitHub

⬡ ALPHA-FIXER

Batch transparency-correction tool for pixel art assets. Normalises alpha channels across sprite sheets to meet GBC pipeline requirements, preventing rendering artefacts caused by inconsistent transparency values.
